# Calculator

## HTML 계산기 실습
### Developed by [EGL](https://github.com/eungyolee) (선린인터넷고등학교 소프트웨어과 1학년 6반 14번 이은교)

#### 1. 제목
- 그냥 평범한 계산기

#### 2. 개발 동기
- 제 뇌가 한계에 부딪혀서 계산기록 저장 기능밖에 떠오르지 않았습니다.

#### 3. 개발 내용
- 계산 기록을 local storage에 저장
- input 태그 안에 직접 수식 입력
- 각 숫자와 연산자 버튼을 클릭하여 입력
- 계산이 완료되었다면 식 초기화
